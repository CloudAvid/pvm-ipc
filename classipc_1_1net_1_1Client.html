Detailed Description

This class is wrapper around Client-sided socket.

In all used syscalls in this class; as error occurrence ,logs will be printed in debug or warning mode includes errno message; unless an exception is thrown.

Warning
after creating a socket, you MUST close it explicity.
Note
errno will be set after failure of any method who uses syscalls.
